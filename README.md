# Awaitsi Website

Building Technology with Heart, for Generations to Come.

## About Awaitsi

Awaitsi is a family-owned South African technology company specializing in:
- Web Development
- Mobile Applications
- CRM Solutions
- AI Integration

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Installation

\`\`\`bash

\`\`\`bash

\`\`\`
